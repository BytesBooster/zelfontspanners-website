## üì∑ Foto's Toevoegen

### Methode 1: Drag & Drop (Aanbevolen)
1. **Sleep foto's** vanuit je computer naar het upload gebied (het gebied met "Sleep foto's hierheen")
2. Je kunt **meerdere foto's tegelijk** uploaden
3. De foto's worden automatisch voorvertoond
4. Geef optioneel een **titel** aan je foto's (geldt voor alle geselecteerde foto's)
5. Klik op **"Foto's Toevoegen"**

### Methode 2: Bestand Selecteren
1. Klik op **"Kies Bestanden"** of het upload gebied
2. Selecteer een of meerdere foto's uit je computer
3. De foto's worden voorvertoond
4. Geef optioneel een **titel** aan je foto's
5. Klik op **"Foto's Toevoegen"**

### Foto Vereisten:
- **Formaat:** Alleen JPEG/JPG bestanden
- **Grootte:** Maximaal 5MB per foto
- **Aantal:** Je kunt meerdere foto's tegelijk uploaden
- **Compressie:** Foto's worden automatisch gecomprimeerd voor betere prestaties

‚ö†Ô∏è **Let op:** Vanwege browser opslag beperkingen kun je ongeveer 10-20 foto's uploaden. Als je een foutmelding krijgt bij uploaden, is de opslag vol.

### Wat gebeurt er na het Uploaden?
- Foto's worden automatisch toegevoegd aan je portfolio
- Ze verschijnen onderaan je foto lijst
- Je kunt ze daarna sorteren (zie hieronder)

## üóëÔ∏è Foto's Verwijderen

### Een Foto Verwijderen
1. Klik op de **üóëÔ∏è (verwijder) knop** onder de foto die je wilt verwijderen

## üîÑ Foto's Sorteren

### Foto's Herschikken (Drag & Drop)
1. **Sleep een foto** naar de gewenste positie
2. Laat de foto los op de nieuwe positie
3. De volgorde wordt automatisch opgeslagen
4. De nieuwe volgorde is direct zichtbaar in je portfolio

### Hoe Werkt het?
- De eerste foto die je ziet is de eerste in je portfolio
- Je kunt foto's naar boven of beneden verplaatsen
- De volgorde wordt automatisch opgeslagen in je browser

### Tips voor Sorteren:
- Plaats je beste foto's bovenaan
- Groepeer gerelateerde foto's bij elkaar
- Je kunt altijd later nog sorteren

## ‚ùì Veelgestelde Vragen

### Kan ik mijn wachtwoord wijzigen?
Op dit moment is het wachtwoord voor alle leden hetzelfde (`test123`). Neem contact op met het bestuur als je je wachtwoord wilt wijzigen.

### Waar blijven mijn foto's opgeslagen?
‚ö†Ô∏è **BELANGRIJK:** Je foto's worden momenteel opgeslagen in je browser (localStorage). Dit heeft belangrijke beperkingen:

**Beperkingen:**
- **Beperkte Opslagruimte:** Je kunt ongeveer 10-20 foto's uploaden voordat de opslag vol raakt (5-10MB limiet)
- **Alleen op deze Computer:** Foto's zijn alleen zichtbaar op deze computer en browser
- **Niet Gedeeld:** Andere bezoekers zien je ge√ºploade foto's niet
- **Kan Verloren Gaan:** Als je je browser cache wist, gaan foto's permanent verloren
- **Geen Back-up:** Er is geen manier om verloren foto's terug te halen

**Wat betekent dit?**
- Foto's blijven bewaard zolang je dezelfde browser gebruikt
- Als je je browser cache wist, kunnen foto's verloren gaan
- Foto's zijn alleen zichtbaar op de computer waar je ze hebt ge√ºpload
- Andere bezoekers zien geen door jou ge√ºploade foto's

**We werken aan een betere oplossing** waarbij foto's permanent en gedeeld worden opgeslagen.

### Kan ik foto's van mijn telefoon uploaden?
Ja! Je kunt foto's uploaden vanaf elk apparaat (telefoon, tablet, computer) zolang je ingelogd bent.

### Hoeveel foto's kan ik toevoegen?
‚ö†Ô∏è **Belangrijk:** Vanwege browser opslag beperkingen kun je momenteel ongeveer **10-20 foto's** uploaden voordat de opslag vol raakt. Als je een foutmelding krijgt ("QuotaExceededError"), is de opslag vol. We werken aan een betere oplossing zonder deze beperkingen.

### Kan ik foto's verwijderen die ik per ongeluk heb toegevoegd?
Ja, je kunt foto's altijd verwijderen via de verwijder knop onder elke foto.

### Waarom zie ik mijn foto's niet in mijn portfolio?
- Controleer of je ingelogd bent
- Refresh de pagina (F5 of Ctrl+R)
- Controleer of de foto's correct zijn ge√ºpload

### Kan ik foto's van andere leden zien?
Ja, je kunt alle portfolio's bekijken via de "Leden" pagina, maar je kunt alleen je eigen portfolio beheren.

### Wat gebeurt er als ik uitlog?
- Je blijft ingelogd voor 24 uur
- Na 24 uur moet je opnieuw inloggen
- Je kunt handmatig uitloggen via het menu
